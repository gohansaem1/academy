# 학생 테이블에 결제일 필드 추가

## 개요
학생 등록 시 매월 결제일을 설정하고, 수업 배정 시 자동으로 결제 항목이 생성되도록 변경되었습니다.

## SQL 마이그레이션

Supabase SQL Editor에서 다음 스크립트를 실행하세요:

```sql
-- students 테이블에 매월 결제일 필드 추가
ALTER TABLE students ADD COLUMN IF NOT EXISTS payment_due_day INTEGER CHECK (payment_due_day >= 1 AND payment_due_day <= 31);
```

## 변경 사항

1. **학생 등록/수정 페이지**
   - 매월 결제일 입력 필드 추가 (1-31일)
   - 기본값: 25일

2. **수업 배정 시 자동 결제 생성**
   - 학생을 수업에 등록하면 자동으로 결제 항목 생성
   - 결제 금액: 수업의 수강료(tuition_fee)
   - 결제일: 학생의 payment_due_day 기준으로 계산
   - 상태: 기본값 'pending' (대기)

3. **결제 등록 페이지 제거**
   - 수강료 관리 페이지에서 "결제 등록" 버튼 제거
   - 결제는 수업 배정 시 자동으로 생성됨

## 사용 방법

1. 학생 등록 시 매월 결제일을 입력합니다 (예: 25일)
2. 수업에 학생을 배정하면 자동으로 결제 항목이 생성됩니다
3. 수강료 관리 페이지에서 결제 상태를 확인하고 관리할 수 있습니다
